# Shortcut Quick Help

* ⌃⌥ ↓ : Move to next screen

* ⌃⌥ ↑ : Move to previous screen

* ⌃⌥ [ : Window to the left screen

* ⌃⌥ ] : Window to the right screen

* ⌃⌥ H : Center Two Thirds

* ⌃⌥ M : Maximize

* ⌃⌥ 1 : Left Third

* ⌃⌥ 2 : Center Third

* ⌃⌥ 3 : Right Third

* ⌃⌥ B : Bottom Half

* ⌃⌥ T : Top Half

* ⌃⌥ C : Center Window

* ⌃⌥ \ : Reset Window

* ⌃⌥ W : Scale up window

* ⌃⌥ S : Scale down window

* ⌃⌥ D : Scale up right side of the window

* ⌃⌥ A : Scale up left side of the window

* ⌥ A : Move window left

* ⌥ D : Move window right

* ⌥ W : Move window up

* ⌥ S : Move window down

* ⌃⌥ / : None
